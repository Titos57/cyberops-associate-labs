# Cisco CyberOps Associate - Lab Portfolio
 
## About
Hands-on labs and research exercises completed as part of the
Cisco CyberOps Associate certification course.
Goal: Build foundational skills for a SOC Analyst Job.
 
**Course Provider:** Cisco Networking Academy
**Status:** In Progress
 
---
 
## Documentation Tiers
 
Labs in this repository are documented at one of three tiers, chosen based on the security relevance and depth of each lab:
 
- **Tier 1** — Full documentation (showcase piece): Overview, Background, Environment & Tools, full narrated Procedure Summary, Key Findings, Security Implications, References.
- **Tier 2** — Medium documentation: Overview, Background, Environment & Tools, condensed Procedure Summary, Key Findings, References.
- **Tier 3** — Brief documentation: Overview, Environment & Tools, one-paragraph Procedure Summary, References.
